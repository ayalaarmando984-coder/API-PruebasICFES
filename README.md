# Ranking ICFES por Municipio

consulta los resultados de las **Pruebas Saber 11 (ICFES)** por institución educativa y municipio, usando la API SODA de datos abiertos de Colombia.

## Link

Original URL: [https://codepen.io/Armando-Ayala-the-looper/pen/emdNZWz](https://codepen.io/Armando-Ayala-the-looper/pen/emdNZWz).

## Funcionalidades

- Selección de año (2014 – 2024)
- Consulta en tiempo real a la API SODA de [datos.gov.co](https://datos.gov.co)
- Agrupación de instituciones por municipio
- Cálculo de promedio de rendimiento (A+, A, B, C, D → escala numérica)
- Ranking ordenado de mejor a peor municipio
- Barra de progreso visual por municipio

## Dataset utilizado

| Campo | Valor |
|---|---|
| Nombre | Pruebas ICFES - Clasificación por municipio e institución |
| Fuente | [datos.gov.co](https://www.datos.gov.co/resource/hk5x-635y.json) |
| Registros | 63 instituciones educativas |
| Cobertura | Risaralda, Colombia · 2014–2024 |
| Formato API | SODA 2 (JSON) |
